# docs/

Put this project's supporting documents here — exported reports, sample
scripts, config files, PDFs, or anything else you want visitors to be able
to open or download alongside the write-up.

**Referencing a file from this folder's `index.html`:**

```html
<a href="docs/remediation-guidelines.pdf">Download remediation guidelines (PDF)</a>
```

No `../` needed — the file and the HTML file are in the same project folder.

Delete this README once the folder has real content in it (or leave it, it
won't break anything).
